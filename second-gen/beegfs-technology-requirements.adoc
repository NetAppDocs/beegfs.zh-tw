若要在 NetApp 上實作 BeeGFS 解決方案、請確保您的環境符合本文件所述的技術需求。



== 需求概述

本頁面詳細介紹了在 NetApp 建置模組上實現 BeeGFS 的硬體、軟體和韌體要求。這些要求分為以下幾個部分：

* <<硬體需求>> - 檔案節點（硬體 / 軟體 / 韌體）、區塊節點（硬體 / 韌體）、網路和佈線
* <<軟體部署需求>> - Ansible 部署的元件
* <<Ansible控制節點需求>> - 控制節點軟體版本




== 硬體需求

在開始之前，請確保您的硬體符合以下針對 BeeGFS on NetApp 解決方案的單一建置區塊設計的規格。特定部署的確切元件可能會因客戶需求而有所不同。每個檔案節點都應符合或超過這些規格，以達到預期效能。
